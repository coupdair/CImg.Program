different type of display for image

3 types of display in this software:

  • 2D map (default)
  • 1D graph (–1D option)
  • 3D surface (–3D option)

others display command line options using -h option

2D map (default)
display image as map
* ./CImg.display -i image.PNG
*
GUI keyboard options:
  • Ctrl+S: save current view
1D graph
display image as both map and 1D graph(s) (either X axis or Y axis)
* ./CImg.display -i image.PNG --1D
*
GUI keyboard options:
  • c: color
  • v: value
  • l: luminance
  • i: intensity
  • Ctrl+S: save current view
GUI mouse options:
  • left button: X axis
  • right button: Y axis
3D surface
display image as 3D surface (e.g. Gouraud-shaded or isophotes render)
* ./CImg.display -i image.PNG --3D
*
GUI keyboard options:
  • F: toggle Fullscreen
  • Q or ESC: Quit (i.e. exit)
load or save file:
  • S: Save displayed image (i.e. 3D view)
  • O: save pOse (i.e. 3D view parameters)
  • R: Read pose (i.e. 3D view parameters)
render type:
  • C: color render (image or constant)
  • T: poinTs render
  • L: Lines render
  • A: fAces render
  • H: flat-sHaded faces render
  • G: Gouraud-shaded faces render
  • P: Phong-shaded faces render
  • I: Isophotes render
  • BackSpace : change rendering type (i.e. decrement type)
  • any other key : change rendering type (i.e. increment type)
GUI mouse options:
  • left button: rotate
  • middle button: move
  • wheel or right button: zoom in/out
